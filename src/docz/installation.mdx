# 安装教程

## 在线安装

#### npm 安装

```js
    npm i @mmcweb/mmc-ui-react
```

#### yarn 安装

```js
    yarn add @mmcweb/mmc-ui-react
```

推荐使用 npm 或者 yarn 的方式安装，它能更好地和 [webpack](https://webpack.js.org/) 打包工具配合使用。
